var is function scoped. It can be re-declared and update and older way to declare variables.
let is block scoped.It Can be updated but not re-declared in the same scope.It is introduced in ES6.
Const block scoped like let.It cannot be re-assigned or re-declared.It must be initialized when declared.


The spread operator expands elements of an array or object into individual elements, making it easier to copy, merge, or pass values.


Map is used to modify or transform each element of an array.It returns a new array of the same length.
Filter is used to select elements that meet a condition.It returns a new array with filtered elements.
forEach() is used to run a function on each element.It does not return a new array.


An arrow function is a compact way to write functions in JavaScript using => syntax.


Template literals are strings written with backticks ( ) that allow variable interpolation (${}) and multi-line strings easily in JavaScript.
